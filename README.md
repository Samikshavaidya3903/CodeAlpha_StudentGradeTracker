# CodeAlpha_StudentGradeTracker
This is a simple student's grade tracker project developed as a part of the CodeAlpha internship. 

## Features
- Add students
- View them all
- Show summary (Average / Highest / Lowest)
- Delete student

## Technologies Used
- Java
- VS Code

## How to Run

1. Compile program

```bash
javac StudentGradeTracker.java
```

2. Run program

```bash
java StudentGradeTracker
```
